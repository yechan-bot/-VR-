# Sunmoon-Dormitory-VR
- vr로 선문대학교 성화학숙을 둘러볼 수 있도록 만든 프로젝트
- 개발기간 : 2020/03 ~ 2020/05
- 개발인원 : 3명

# 주제 선정 이유 및 목적
- 다른 시스템과 차별되는 입체적인 기숙사 시각 정보 제공
- 가상공간을 통한 간접 체험
- 기숙사 입주에 도움을 주기 위함

# 유사 어플리케이션 (다방, 직방) 과의 차이점
- VR 서비스를 통해 사용자 본인이 직접 보는 듯한 체험 제공
- 교내 기숙사 정보를 제공

# 개발환경
- SketchUp
  - 기숙사 3D 모델링
- Unity
  - 가상공간 구현
- Android Studio
  - UI 구현
